      - Store in `markers` Map keyed by camera.id

| T-6-05 | Information Disclosure | Camera GPS in page props | accept | Dashboard is behind auth+verified middleware. Camera coordinates are operational data for authorized operators. |
| T-6-06 | Denial of Service | Rapid pulse ring creation | accept | CSS animations are lightweight. animationend listener removes elements. Bounded by camera event rate (~1/second max per camera). |
