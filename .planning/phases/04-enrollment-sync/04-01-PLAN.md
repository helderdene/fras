    - Load personnel records: `Personnel::whereIn('id', $personnelIds)->get()`
    - Chunk into batches of `config('hds.enrollment.batch_size')`
    - For each chunk, generate messageId: `'EditPersonsNew' . now()->format('Y-m-d\TH:i:s') . '_' . Str::random(6)`
    - Build payload via `buildEditPersonsNewPayload($camera, $chunk, $messageId)`
    - Store cache entry: `Cache::put("enrollment-ack:{$camera->id}:{$messageId}", ['camera_id' => $camera->id, 'personnel_ids' => $chunk->pluck('id')->toArray(), 'photo_hashes' => $chunk->pluck('photo_hash', 'custom_id')->toArray(), 'dispatched_at' => now()->toIso8601String()], config('hds.enrollment.ack_timeout_minutes') * 60)`
    - Publish MQTT: `MQTT::publish("mqtt/face/{$camera->device_id}/Edit", json_encode($payload))`

| T-4-06 | Information Disclosure | Cache correlation entries | mitigate | Cache entries use non-guessable messageId (timestamp + Str::random(6)); TTL auto-expires |
